## Scala ActiveRecord sample

run sbt.

```sh
$ bin/sbt
```

### Console application samples

```
> sample/run
```

### Scalatra sample

```
> scalatra/container:start
```

The application starts on http://localhost:8080.

### Play! 2.x sample

```
> play2x/run
```

The application starts on http://localhost:9000.
